MediaPipe

MediaPipe is an open-source framework developed by Google for building machine learning pipelines,
particularly for processing time-series data such as video, audio, and text. It is designed to work across
multiple platforms, including desktops, mobile devices (Android and iOS), and embedded systems
like Raspberry Pi and Jetson Nano, and in this case, Arduino.

OpenCV

OpenCV (Open Source Computer Vision Library) is an open-source computer vision and machine learning
software library. It is widely used for various image and video processing tasks, such as object detection,
face recognition, and image transformation. OpenCV supports multiple programming languages, including Python,
C++, and Java.

The Code, what makes Serial Comunication Awesome

What is Serial Communication(In Arduino)?

Serial communication in Arduino facilitates data exchange between the Arduino board
and external devices such as computers, sensors, or other microcontrollers.
It is a fundamental feature of Arduino, enabling debugging, data logging, and interaction with peripherals, In this case, Python communicates with the Arduino through a wireless bluetooth module (Esp32). Python sends the information, the Arduino IDE recieves it and processes it, and finally the Arduino executes it
